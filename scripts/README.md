# scripts

This directory will contain project automation scripts in later milestones.

Planned script categories:

- **Ingestion**
  - parse/validate content
  - chunk and embed corpus
  - upsert vectors and generate ingestion reports

- **Smoke testing**
  - basic API/runtime checks
  - retrieval sanity checks

- **Evaluation helpers**
  - run question bank
  - record rubric scores and regression comparisons

- **Deployment helpers**
  - pre-deploy checks
  - post-deploy smoke validation
